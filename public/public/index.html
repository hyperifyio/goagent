How it works

1. Pick a model

Use your preferred provider. Swap models without changing your app.

2. Enable tools

Toggle web, code, image and file tools as needed per task.

3. Run

Call the agent with a prompt or plan. Stream output or capture artifacts.

Quick start

  1. Install: run in your project: go get github.com/hyperifyio/goagent@latest
  2. Import: add the package and configure your AI provider.
  3. Run: run your program and enter a prompt.

For complete code, docs, and details, see the README on GitHub.

Frequently asked questions

Is this a full framework?

No | It is a lean LLM shell that lets you compose tools and workflows without heavy abstractions.

Which models are supported?

The design is provider–agnostic; pick models that suit your use case. See the repo for compatibility notes.

Production ready?

Yes. The API is intentionally small with predictable behavior for production usage.
